# Abdullah Raad - Executive Portfolio

A premium, modern SaaS-style personal portfolio for **Abdullah Raad**, an AI Systems Architect, Full Stack Developer, and English Trainer based in Baghdad, Iraq.

This portfolio is highly optimized, fully responsive, and architected utilizing advanced AI workflows via **Lovable**, **Cursor**, and **Replit**.

## 🚀 Built With
- **Frontend Core:** Vanilla HTML5, JavaScript
- **Styling:** Tailwind CSS (via CDN)
- **Typography:** Inter (Google Fonts)
- **Development Engine:** Lovable & Cursor (AI-driven workflow reducing delivery cycles by 60-80%)
- **Deployment Ready:** Vercel

## ✨ Key Features
- **Bento Grid Architecture:** Clean, modular layout for technical projects.
- **Glassmorphism UI:** Deep background blur (`backdrop-blur`), translucent cards, and a semi-transparent glowing hero portrait.
- **Bilingual System (EN/AR):** Seamlessly toggles between English and Arabic (RTL support) with native CSS class logic.
- **'Antigravity' Animations:** Subtle custom Y-axis floating animations that give the interface a high-end, responsive feel.
- **Zero-Dependency Core:** Runs straight out of the box with zero heavy frontend frameworks, relying entirely on raw performance and Tailwind utilities.

## 📦 Deployment Instructions (Vercel)
This repository is production-ready for [Vercel](https://vercel.com/):
1. Push the code to a GitHub repository.
2. Import the repository in your Vercel Dashboard.
3. Vercel will automatically detect the static `index.html`. No build commands are required.
4. Deploy and enjoy your lightning-fast portfolio!

## ⚙️ Content Structure
- `index.html`: The monolithic source file containing all markup, styles, and logic.
- `profile.jpg`: The hero portrait image layered with blend modes.
